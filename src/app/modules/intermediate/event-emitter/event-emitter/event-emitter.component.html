¿Qué es Event Emitter?


Un problema con el que te toparás en un futuro al trabajar con Angular, es encontrar la necesidad de compatir información entre distintos componentes, de un componente padre a un componente hijo, y viceversa. Para este problema, Event Emitter es una solución práctica, que simplifica el proceso de comunicación etre componentes, clases o servicios.

En conceptos generales, Event Emitter consiste en un receptor y un emisor (o varios), en el que el receptor se suscribe a un "EventEmitter", y espera a que los emisores manden una señal.
