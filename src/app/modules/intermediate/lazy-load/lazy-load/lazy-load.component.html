Lazy Load (Carga perezosa)


A continuación se presentará una introducción a Lazy Load o carga perezosa.


¿Qué es Lazy Load?


Al momento de desarrollar una aplicación web, es posible que quieras crear varias vistas, en este caso, componentes, los cuales tu navegador debe cargar. A partir de cierto tamaño del proyecto, a los navegadores se les haría imposible, o al menos, muy lento cargar todos los recursos al mismo tiempo. Es aquí en donde entra Lazy load o carga perezosa, la cual consiste en cargar solo aquellos módulos que sea estrictamente necesario su uso, evitando así cargar todo aquello que no sea estrictamente necesario.

Por ejemplo, en un Login, no es necesario cargar la información de la aplicación, ya que aun ni se tiene acceso a ella. por lo que no tendría sentido cargarla. Del mismo modo, una vez iniciada sesión, el módulo del Login ya no es requerido, y tampoco es necesario cargarlo cada vez que navegues en tu página.

Implementación


Para poder cargar componentes de forma perezosa, se requiere que estos estén vinculados a un módulo, por ende, el primer paso debe ser crar un módulo. Es importante que este módulo cuenta con un módulo de rutas, como se ve en el siguiente comando:


ng generate module lazyexample --routing

Lo siguiente que tienes que hacer, es crear el componente que se cargará y se agregará al archivo de rutas recién creado.


ng generate component mycomponent --module lazyexample

lazyexample-routing.module.ts

{{ routerCode }}

Nota como en el path se encuentra con una cadena vacía, esto debido a que la ruta para acceder al componente se deberá indicar en el archivo "app-routing.module.ts".

app-routing.module.ts


{{ approuterCode }}

Aquí podemos ver como al ingresar la ruta "example", se manda a cargar el módulo "Lazyexample", el cual carga en automático el componente "mycomponent".
