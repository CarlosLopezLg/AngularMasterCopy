Principales comandos en Angular CLI


A continuación se presentará un introducción al archivo de rutas de Angular.

Instalar CLI de Angular


npm install -g @angular/cli

Crear un proyecto nuevo


ng new my-first-project

En donde "my-first-project" será el lugar en donde se insertará el nombre del proyecto.

Correr el proyecto en el navegador


ng serve

Banderas:

-o: Abrir el navegador automáticamente

Compilar proyecto


ng build --configuration production

Con este comando, preparas los archivos para subir a producción la aplicación. También pasarán de Typescript a Javascript los archivos.

Crear componentes


//Comando completo
ng generate component component-name
//Método abreviado
ng g c component-name

Banderas:

--skip-tests: Omite la creación del archivo de pruebas ".spec.ts"

--module: Selecciona el módulo a declarar el componente.

Crear servicio


ng generate service service-name

Banderas:

--skip-tests: Omite la creación del archivo de pruebas ".spec.ts"

Crear módulo


ng generate module module-name

Banderas:

--routing: Crea archivo de enrutamiento

Crear guard


ng generate guard guard-name

Generar comando


Generador:

Selecciona un elemento a generar:

Elemento a generar {{option.name}}

Selecciona un nombre:

Nombre del componente

Opciones extras:

No generar archivo de testing
Incluir en módulo
Generar archivo de rutas

{{ generatedCommand }}
