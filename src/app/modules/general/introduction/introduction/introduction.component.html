Introducción a Angular


A continuación se presentará una introducción y un conjunto de requisitos para empezar a trabajar con este framework.



¿Qué es Angular?


Angular es un framework que nos ayuda a desarrollar aplicaciones web SPA, el cual está orientado a componentes. Si quieres darte una idea de qué se puede hacer con esta herramienta, YouTube es un ejemplo claro, y si aun no estás impresionado, ¡también esta misma página de donde estás leyendo esto está desarrollada en Angular!

Este framework, junto con React y Vue son los más solicitados a nivel laboral, por lo que agregar Angular a tu curriculum no vendría nada mal.


Requisitos


Para poder empezar a programar con este framework, se necesitan cumplir los siguientes requisitos:

Node.js: Puedes decargar la última versión aquí.

Npm: Necesario para la instalación de paquetes, suele estar incluído en la instalación de Node.

Angular CLI: Necesario para correr comandos de Angular. para su instalación es necesario abrir la terminal e insertar el siguiente comando:


npm install -g @angular/cli

Nota: Algunas terminales por default no permiten ingresar comandos de Angular, si presentas este error, puedes solucionarlo con el siguiente comando:

Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned

Comenzar a desarrollar


Para comenzar a desarrollar tu primera aplicación web con Angular, deberás dirigirte en tu terminal a la dirección donde desees crear el proyeco. A continuación, deberás insertar el siguiente comando:


ng new my-first-project

Una vez creado el proyecto, veremos como luce inicialmete el proyecto, sin embargo, debes saber que Angular requiere de otro comando para poder mostrar la página. El comando es el siguiente:

//Ingresar a la carpeta del proyecto
cd my-first-project
//Iniciar proyecto
ng serve -o
