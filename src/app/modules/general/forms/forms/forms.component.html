A continuación se presentarán estructura básica de un formulario.


FormGroup


Los formularios son parte clave de una aplicación web, ya que es la forma en la que un usuario puede ingresar información. Utilizar formularios reactivos tiene múltiples ventajas, como validar los campos de entrada, estandarizar valores y otras características más.

Requiere de tener importado el módulo "ReactiveFormsModule".

Nota: Para explotar mejor el potencial de los formularios reactivos, se trabajará con Angular Materials.
