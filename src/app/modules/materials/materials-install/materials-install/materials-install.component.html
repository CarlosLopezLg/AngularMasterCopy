Introducción a Angular Materials


Angular Materials te da acceso a múltiples herramientas y componentes para poder diseñar aplicaciones web más vistosas de manera sencillo. Materials podría ser comparado con Bootstrap. También, materials realiza ciertas configuración de diseño general, como establecer la fuente 'Roboto' como fuente predeterminada.

Instalación


Para instalar Angular Materials, debes usar el siguiente comando:


ng add @angular/material
